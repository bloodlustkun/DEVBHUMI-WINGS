
# Devbhoomi - Premium Travel & Adventure Experiences

Welcome to Devbhoomi, your gateway to premium travel experiences and adventure trekking in the Himalayas.

## Features

- 🏔️ **Premium Adventure Trekking** - Explore stunning mountain valleys and peaks
- ✈️ **Curated Travel Experiences** - Expertly designed tours and journeys
- 🎬 **Professional Drone Coverage** - Premium aerial photography and videography
- 📱 **Professional Content Strategy** - LinkedIn and social media presence

## About

Devbhoomi specializes in creating unforgettable travel experiences with:
- Expert guides and local knowledge
- Sustainable tourism practices
- Professional content creation
- Personalized itineraries

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

Run `npm run build` to create a production build.

## Technologies Used

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Radix UI** - Component library
